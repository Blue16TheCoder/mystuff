How to do Math in Javascript

Check console.log to see math stuff. Coming in UI soon! Will be merged into javascript.

To do square roots do Math.sqrt(Number). You can do this inside console or make an variable.

To make exponential notation make an variable and put in some big number. Then use that variable to do the method toExponential (NAD). Then put that method in the same variable. Ex: variable.toExponential().

To make an number a specific lenght use variable.toPrecision to specify how much numbers you want. Put the number you want in a variable and use that variable once you used that number.
